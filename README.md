# Lare

> Disminuya el bitrate de sus archivos MP3 desde la terminal

## Descripción

Lare es un pequeño script de Shell para __LAME MP3 Encoder__ creado con la finalidad de automatizar la tarea de disminuir el bitrate de un lote de archivo MP3 desde la terminal.

## Requisitos

Para que este script funcione adecuadamente es necesario seguir los siguientes pasos:

- Tener instalado [LAME MP3 Encoder](https://lame.sourceforge.io/).
- Ubicar el ```Lare.sh``` archivo dentro de la carpeta donde se encuentran todos los archivos MP3 a convertir.
- Asignar los permisos pertinentes a ```Lare.sh``` usando el comando ```chmod +x Lare.sh```
- Proceder a ejecutar el archivo ```Lare.sh```.
